# 调度配置说明

## 触发类型

| 类型 | 说明 |
|------|------|
| 手动触发 | 需要人工触发才会执行 |
| 周期调度 | 按配置周期自动执行 |

## 周期类型

| 周期 | startTime 默认值 |
|------|-----------------|
| 天 | 00:00 |
| 周 | 周一 00:00 |
| 月 | 1号 00:00 |
| 小时 | 每小时整点 |

## 业务基线

- **必须关联**：每个同步任务必须关联一个业务基线
- **作用**：保障数据产出时效，用于 SLA 监控
- **配置**：向用户展示可选业务基线列表供选择

## 其他参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| retryCount | 3次 | 失败重试次数 |
| throttleLimit | 10M/s | 写入限流值 |
