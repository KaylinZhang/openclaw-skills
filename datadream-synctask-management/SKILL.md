# 同步任务管理

## 前置检查

**强制**：第一次调用skill时，先确认 DataDream MCP 服务已安装。未安装则立即结束。

## MCP Tools 工具映射

| 操作 | MCP 工具 |
|------|----------|
| 查询同步任务列表 | `list_sync_tasks` |
| 查询同步任务详情 | `get_sync_task` |
| 删除同步任务 | `delete_sync_task` |
| 启用同步任务 | `enable_sync_task` |
| 停用同步任务 | `disable_sync_task` |
| 下线同步任务 | `offline_sync_task` |

## 查询同步任务列表

**触发**："同步任务列表"、"查看同步任务"、"同步任务有哪些"

| 参数 | 说明 |
|------|------|
| keyword | 按任务名称关键词筛选 |
| status | 按任务状态筛选（运行中/已停用/已下线） |
| owner | 按负责人筛选 |

**流程**：
1. 调用 MCP 工具查询当前项目下的同步任务列表
2. 按筛选条件过滤（如有）
3. 展示任务基本信息：任务名称、类型、状态、负责人、创建时间

---

## 查询同步任务详情

**触发**："同步任务详情"、"看一下某个任务"、"任务详情"

| 参数 | 必填 | 说明 |
|------|------|------|
| id/name | 是 | 任务 ID 或名称 |

**流程**：
1. 调用 MCP 工具根据 ID 或名称查询同步任务详情
2. 展示完整信息：基本信息、来源去向、字段映射、调度配置、告警配置、创建时间

---

## 删除同步任务

**触发**："删除同步任务"、"删除任务"

**前置约束**：运行中的任务不可删除。

**流程**：
1. 查询任务当前信息和状态
2. 检查任务状态：
   - 运行中：提示用户"任务正在运行中，请先停用后再删除"
   - 非运行中：继续下一步
3. 删除前向用户确认删除操作
4. 收到明确确认后才能调用 MCP 工具删除
5. 返回删除结果

---

## 启用同步任务

**触发**："启用同步任务"、"启用任务"

**前置约束**：已停用或已下线的任务才能启用。

**流程**：
1. 查询任务当前状态
2. 校验任务状态是否允许启用
3. 启用前向用户确认（可选，展示当前配置摘要）
4. 调用 MCP 工具启用
5. 返回启用结果

---

## 停用同步任务

**触发**："停用同步任务"、"停用任务"

**前置约束**：运行中的任务才能停用。

**流程**：
1. 查询任务当前状态
2. 校验任务状态是否允许停用
3. 停用前向用户确认（可选）
4. 调用 MCP 工具停用
5. 返回停用结果

---

## 下线同步任务

**触发**："下线同步任务"、"下线任务"

**前置约束**：已停用的任务才能下线。下线后任务不可恢复。

**流程**：
1. 查询任务当前状态
2. 校验任务状态是否允许下线
3. ⚠️ 下线操作不可逆，下线前必须向用户明确说明风险
4. 收到明确确认后才能调用 MCP 工具下线
5. 返回下线结果

---

## 约束与注意事项

- **项目归属**：所有操作默认在当前项目下进行
- **操作不可逆**：下线操作不可恢复，删除操作不可恢复，执行前必须获得用户明确确认
- **状态依赖**：启用→停用/下线→启用；下线后不可再启用
- **运行中保护**：运行中的任务不可删除、不可停用
- **修改与删除要求**：未明确授权可修改的参数均不可修改，未明确授权可删除的参数均不可删除
