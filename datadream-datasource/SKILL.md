# 数据源管理

## 前置检查

**强制**：先确认 DataDream MCP 服务已安装。未安装则立即结束。

## 能力边界

| 操作 | 支持状态 |
|------|---------|
| 创建「数易数据填报」数据源 | ✅ |
| 查询数据源列表/详情 | ✅ |
| 修改数据源 | ✅（仅名称） |
| 删除数据源 | ✅（需无关联任务） |
| 创建其他类型（MySQL、Kafka等） | ❌ 待扩展 |

## 数据源模型

```
DataSource
├── id          # 系统生成
├── name        # 名称（不填自动生成）
├── type        # 数据源类型
├── config      # 类型特定配置
└── projectId   # 所属项目（默认当前项目）
```

## 创建数据源

**触发**："创建数据源"、"新增数据源"、"注册数据源"

### 类型「数易数据填报」（类型标识：`datae_dataentry`）

**用途**：作为同步任务来源，读取数易项目空间下的数据填报表。

| 参数 | 必填 | 说明 |
|------|------|------|
| name | 否 | 不填时自动生成：`{项目空间名称}_数据填报` |
| shuyiProjectSpace | 是 | 绑定的数易项目空间 |

**流程**：
1. 收集项目空间信息（未提供则查询列表供选择）
2. 确认名称和项目空间
3. 调用 MCP 创建
4. 返回结果

**约束**：
- DataDream 和数易必须同环境
- 名称唯一（同一项目下）
- 类型不可变

## 查询数据源

**触发**："数据源列表"、"查看数据源"

| 参数 | 说明 |
|------|------|
| type | 按类型筛选 |
| keyword | 按名称关键词筛选 |

## 修改数据源

**触发**："修改数据源"、"编辑数据源"

仅支持修改名称。流程：查询当前信息 → 收集修改内容 → 确认 → 调用 MCP → 返回结果。

## 删除数据源

**触发**："删除数据源"

**前置约束**：数据源下无关联同步任务方可删除。

流程：查询信息 → 检查关联任务 → 如有关联则列出并终止 → 确认 → 删除。
